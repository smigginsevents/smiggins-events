# Sound Files

Drop the following audio files in this folder to enable sound effects during the live show.
All files are optional — missing files are silently skipped.

| File | Used for |
|---|---|
| `tick.mp3` | Looping tick sound during the countdown timer |
| `buzzer.mp3` | Played when the timer hits zero |
| `drumroll.mp3` | Played when transitioning to the leaderboard |
| `fanfare.mp3` | Played at the final winner reveal |
| `reveal.mp3` | Short sting played when the answer is revealed |

**Recommendations:**
- Keep files short (tick: ~1s loop, buzzer: ~1s, drumroll: ~3s, fanfare: ~4s, reveal: ~1s)
- 128kbps MP3 is fine
- Free sources: freesound.org, pixabay.com/music, zapsplat.com
